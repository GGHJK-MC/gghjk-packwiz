# GGHJK Packwiz
This modpack is for the [GGHJK](https://www.gghjk.qzz.io) server.

## Installation
Download the Prism Launcher.

Click on "Add Instance", then "Import".

Paste in the following link and click "OK".

```
https://github.com/MC-GGHJK/gghjk-packwiz/releases/download/1.0.0/GGHJK.zip
```

Launch the imported instance by double-clicking on it. You may be prompted to sign in to your Minecraft account.
